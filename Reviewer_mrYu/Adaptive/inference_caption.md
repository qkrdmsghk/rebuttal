# Figure S_: Detailed Inference Pipeline of AdaPert's Adaptive Subgraph Extraction

**Detailed illustration of AdaPert's adaptive learning step at inference for an unseen perturbation gene $p$.** (Top-left) The perturbation gene $p$ is located on a shared KG template $\mathcal{G}$ (STRING PPI network). A 4-layer GATv2 with skip-cat connections produces structural node embeddings $\mathbf{h}_i \in \mathbb{R}^{128}$ for all genes, while a pre-trained language model (GenePertESM) provides semantic embeddings $\mathbf{s}_i \in \mathbb{R}^{3072}$. (Middle) A subgraph scoring function $\sigma(\mathbf{W}_c[\mathbf{h}_i \| \tilde{\mathbf{s}}_i])$ computes perturbation-conditioned attention weights $\alpha_i$ for each neighbor of $p$ by jointly leveraging structural and semantic features. Neighbors exceeding a threshold $T$ are selected to form the context subgraph $\mathcal{G}_{\text{context}}$ (top-right), which adaptively varies in size per perturbation. (Bottom) The encoder $\text{ENC}_\theta$ maps control cell expression $\bar{\mathbf{x}}_c$ to a basal state representation $\mathbf{z}_c$. The context vector $\mathbf{z}_p = \sum_{v \in \mathcal{G}_{\text{context}}} \alpha_v \cdot \mathbf{h}_v$ is computed via attention-weighted sum pooling over selected neighbors. The decoder $\text{DEC}_\phi$ takes the concatenation $[\mathbf{z}_c, \mathbf{z}_p, \mathbf{s}_p]$ and predicts perturbed gene expression $\hat{\mathbf{x}}_{\text{pert}}$. Crucially, the GNN and subgraph scoring operate on a fixed KG template shared across all perturbations — the adaptive step is purely attention-based, requiring no perturbation-specific graph construction at inference time.
