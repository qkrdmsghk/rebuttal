Attention-based threshold analysis for adaptive subgraph extraction (POLR2B perturbation, Layer 3 of 4-layer GATv2). For each attention threshold T ∈ {0.05, 0.08, 0.10, 0.20}, red bars show the number of high-attention nodes (max head attention > T) within the perturbation gene's k-hop STRING neighborhood (1-hop to 4-hop). Blue dashed lines indicate the number of ground-truth DEGs at each hop distance. Rightmost panel: Pearson Delta performance across thresholds. At T=0.05, nearly all neighbors are retained (100%); at T=0.10, the selected node count approaches the DEG count at 1-hop (11 vs. 8) and 2-hop (42 vs. 67), coinciding with peak performance (Pearson Delta = 0.6199). At T=0.20, excessive filtering removes too many informative nodes and performance decreases. This analysis motivates the threshold range [0.05, 0.20] and identifies T=0.10 as optimal.
