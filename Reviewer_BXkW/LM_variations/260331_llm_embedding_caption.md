**Figure X. Sensitivity of perturbation prediction to the choice of language model for gene summary generation.**
**(a)** Test Pearson delta comparison between GPT-4o and Claude (Haiku 4.5) for generating gene functional summaries, which are subsequently embedded using the same text-embedding-3-large model (3,072 dimensions). Two reference source conditions are tested: NCBI + UniProt (combining gene descriptions with protein functional summaries) and NCBI only (gene descriptions alone). Delta values indicate the performance difference (Claude − GPT). **(b)** Composition of gene summaries by source. When NCBI + UniProt references are available, 99.4% of the 18,166 genes in the STRING PPI network have existing database descriptions, leaving only 117 genes (0.6%) requiring AI generation. With NCBI descriptions alone, 1,364 genes (7.5%) lack substantive descriptions and require AI-generated summaries. The performance difference between LLMs scales with the proportion of AI-generated content: negligible when most summaries come from curated databases, but measurable (+0.0050 Pearson delta) when a larger fraction relies on AI generation.
